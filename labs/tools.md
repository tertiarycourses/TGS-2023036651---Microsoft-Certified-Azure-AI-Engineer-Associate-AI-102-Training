# Tools Guide

## Required Tools

| Tool | Purpose |
| --- | --- |
| Azure portal | Provision and manage AI resources. |
| Azure AI Foundry | Build generative AI, model, prompt flow, and agent solutions. |
| Azure Cloud Shell | Run Azure CLI commands. |
| Visual Studio Code | Edit code, REST files, and JSON payloads. |
| REST client or Postman | Test APIs and endpoints. |
| Python or C# SDK examples | Integrate AI services into applications. |

## Optional Tools

| Tool | Purpose |
| --- | --- |
| Azure AI Search index explorer | Test search and vector queries. |
| Azure Storage Explorer | Inspect source documents and blob data. |
| diagrams.net | Draw AI solution architectures. |

## Cleanup Checklist

At the end of class, check for:

- Foundry projects and model deployments.
- Azure AI services resources.
- Azure AI Search services and indexes.
- Document Intelligence resources.
- Storage accounts with training documents.
- Endpoints, keys, and sample applications.
- Resource groups created only for training.
