Hands-on Azure AI Engineer labs for learners building practical AI solutions with Microsoft Azure and Microsoft Foundry services. The labs cover Azure AI solution planning, responsible AI, Microsoft Foundry resources, generative AI, Azure OpenAI, prompt flow, RAG, agents, computer vision, custom vision, video insights, NLP, speech, translation, Azure AI Search, Document Intelligence, Content Understanding, monitoring, security, and course review.

## How to Use These Labs

1. Read the learner guide before starting the first lab.
2. Complete the labs in order because later exercises reuse the same Azure AI solution architecture.
3. Use Microsoft Learn, an Azure free account, or an instructor-provided Azure subscription.
4. Prefer low-cost resources and free tiers where available.
5. Delete model deployments, indexes, endpoints, and resource groups created only for training.
